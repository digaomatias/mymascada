# Design Context — MyMascada

## Target Audience
Young professionals and families managing personal finances in New Zealand. Tech-savvy but not necessarily design-literate. Ages 25-45.

## Use Cases
- Track daily spending and income
- Connect bank accounts and auto-categorize transactions
- Set and monitor financial goals (savings, budgets)
- AI-powered financial insights via chat
- Dashboard overview of financial health

## Brand Personality / Tone
- **Clean, modern, trustworthy** — it's a finance app, people need to trust it with their money data
- **Warm but professional** — not cold/corporate, not playful/childish
- **Confident and calm** — like having a smart friend who's good with money
- Think: Notion meets Monzo — refined simplicity with personality

## Visual Direction
- Prefer refined minimalism with subtle warmth
- Avoid generic fintech blue — use tinted neutrals with a distinctive accent color
- Dark mode should feel premium, not just inverted
- Typography should feel modern and readable, not generic
- Cards are fine but avoid nesting; use spacing and hierarchy instead

## Tech Stack
- Next.js 15 + React 19
- Tailwind CSS v4
- Radix UI primitives
- Heroicons
